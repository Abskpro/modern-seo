Instructions for deployment

Setup a Heroku account

SEO is all about how the outside world sees your web application, so can't really do much without first deploying to a server somewhere.

Make sure you begin by installing Heroku toolbelt, and then run

      
  heroku login

to login with your heroku credentials, and then finally...

      
  heroku apps:create [app-name]
  heroku config:set EXERCISE=deploy

to set up a new app. Now you just have to push to the heroku git remote to trigger a deploy

      
  git push heroku master

You can open your app, as hosted on heroku by running

      
  heroku open
